# Usage 

1. Initialize your project

```sh
$ ./new.sh yourProjectName
```

2. install dependencies 
```sh 
$ make install
```

3. make project files 

```sh 
$ make project
```

4. remove new.sh 

```sh 
$ rm -rf new.sh
```
